```
程序说明:

1.启动服务go run main.go

2.访问localhost:8888/start，调用一次加一个job，直到job队列满之后，job不再增加;
  job开始标志是在控制台打印当前时间。

3.访问localhost:8888/stop，调用一次结束一个job，直到job队列清空，所有goroutine停止;
  job停止标志是控制台停止打印当前时间

```

```
4，（编程，可以选择不作答）创建运行（Start）有且仅有一个任务线程，关闭任务线程（Stop），要求：
一，Start或Stop支持多线程安全调用；
二，调用Start后，运行有且仅有一个任务线程，任务线程运行后，等待接受任务，
三，当接受到任务后，打印出已经接受的任务数量；然后再次等待接受任务
四，如果Start时线程已启动，那么不做处理，
五，调用Stop后，会关闭并退出线程，已接受的任务数量不清零
```
